# Setting up F3Set

This directory contains the splits and labels from F3Set. To download the trimmed videos frames and generate the folder structure for frames, follow the instructions provided in F3Set: https://github.com/F3Set/F3Set/tree/main/data/f3set-tennis

Frames are extracted at a resolution of 796x448 (WxH), following ths frame naming convention:

```
data-folder
└───F3SetFrames
    └───20120628-M-Wimbledon-R64-Lukas_Rosol-Rafael_Nadal_101065_101157
        |000000.jpg
        |000001.jpg
        |...
        |000091.jpg
    └───20120628-M-Wimbledon-R64-Lukas_Rosol-Rafael_Nadal_101885_102145
        |000000.jpg
        |000001.jpg
        |...
```
